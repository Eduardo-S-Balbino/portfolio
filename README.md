# Portfolio

Professional portfolio website developed to present my profile, skills, and projects as a Backend Developer in training.

## Live Website

Access the online portfolio here:

[https://portfolio-ekgq.onrender.com](https://portfolio-ekgq.onrender.com)

## About the Project

This project was created to showcase my professional profile, technical skills, and practical projects in a clear and modern way.

The portfolio highlights my focus on backend development with Python and presents my main projects, technologies, and contact information.

## Features

- Responsive portfolio website
- Modern interface with dark/light mode toggle
- Sections for About, Skills, Projects, and Contact
- External links to GitHub, LinkedIn, and live project
- Google Analytics 4 integration
- Custom event tracking for user interactions

## Tracked Events with Google Analytics

This portfolio includes Google Analytics 4 to monitor user interaction.

Tracked events include:

- `page_view`
- `click_portfolio_button`
- `theme_toggle`

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Google Analytics 4
- Render
