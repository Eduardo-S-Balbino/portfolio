Projeto em destaque

People Management Web

Aplicação web completa desenvolvida com Flask para gerenciamento de pessoas. O sistema possui autenticação de usuários, CRUD completo, busca, paginação, integração com banco de dados e deploy online.

  • Autenticação de usuários com login e logout
  • CRUD completo para cadastro, edição e remoção de pessoas
  • Busca por nome e paginação de resultados
  • Banco SQLite no desenvolvimento e PostgreSQL em produção
  • Estrutura modular com Blueprints e camada de serviço
  • Deploy realizado na Render
